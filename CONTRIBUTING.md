# Contributing to AddictionToFreedom

Thank you for your interest in contributing. This project is built to help people, and every improvement matters.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/your-username/addiction-to-freedom.git`
3. Install dependencies: `npm install`
4. Copy env file: `cp .env.example .env.local`
5. Start dev server: `npm run dev`

## Branch Naming

- `feature/your-feature-name`
- `fix/what-you-fixed`
- `docs/what-you-updated`

## Pull Requests

- Keep PRs focused — one feature or fix per PR
- Write clear commit messages
- Ensure `npm run lint` passes before submitting
- Describe what you changed and why

## Code Standards

- TypeScript strict mode — no `any` types
- Tailwind CSS for styling
- Accessible markup (WCAG 2.1 AA)
- Compassionate, inclusive language in all copy

## Questions

Open an issue or start a discussion on GitHub.
